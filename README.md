# Nissan Leaf

Adds support for Nissan Leaf pre May 2019

## Local development

Prerequisites:
- [Node.js and NPM](https://nodejs.org/en/download)
- [git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
- Homey (npm i -g homey)

```sh
npm install
homey app run
```

Be sure to follow eslint
```sh
npm run lint-fix
```
